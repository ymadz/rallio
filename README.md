Badminton Court Finder & Queue Management System for Zamboanga City, Philippines

## Project Structure

- `docs/` - All documentation
- `backend/` - Supabase backend (PostgreSQL + APIs)
- `web/` - Next.js 14 web application
- `mobile/` - React Native mobile application (Expo)
- `shared/` - Code shared between web and mobile
- `scripts/` - Utility scripts

## Quick Start

### Web App
```bash
cd web
npm install
npm run dev
```

### Mobile App
```bash
cd mobile
npm install
npx expo start
```
